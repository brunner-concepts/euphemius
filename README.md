# Euphemius

Euphemius is an early web presence and product shell for stablecoin payment rails focused on African and global trade corridors.

## Structure

- `site/` is the public marketing site for `euphemius.com` and `www.euphemius.com`.
- `app/` is the private alpha app shell for `app.euphemius.com`.

Both folders are static Cloudflare Pages targets and require no build step.

The hero visual uses a public-domain NASA Earth Observatory image hosted by Wikimedia Commons:
`https://upload.wikimedia.org/wikipedia/commons/thumb/2/2e/City_Lights_of_Africa%2C_Europe%2C_and_the_Middle_East2.jpg/2048px-City_Lights_of_Africa%2C_Europe%2C_and_the_Middle_East2.jpg`

## Cheapest Launch Setup

Use one Google Workspace Business Starter seat:

- Paid mailbox: `agranda@euphemius.com`
- Optional alias later: `brunner@euphemius.com`
- Do not delete Microsoft or existing mail records until the current mail path has been inventoried and Google mail is verified.

Cloudflare DNS records needed after Google Workspace setup:

- Google verification TXT record from Google Admin.
- `@ MX 1 smtp.google.com`
- `@ TXT "v=spf1 include:_spf.google.com ~all"`
- `google._domainkey TXT <value generated by Google Admin>`
- `_dmarc TXT "v=DMARC1; p=none; rua=mailto:agranda@euphemius.com"`

Keep DMARC at `p=none` during launch so mail can be monitored before enforcing stricter policy.

## Cloudflare Pages

Create two Pages projects from this repository:

- `euphemius-site`
  - Root directory: `site`
  - Build command: none
  - Output directory: `/`
  - Custom domains: `euphemius.com`, `www.euphemius.com`

- `euphemius-app`
  - Root directory: `app`
  - Build command: none
  - Output directory: `/`
  - Custom domain: `app.euphemius.com`
